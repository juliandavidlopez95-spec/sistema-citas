En este documento podrá encontrar la información general sobre el componente front-end de mi proyecto.

## Descripción

Este módulo corresponde a una versión temprana del portal web del **Sistema de Gestión de Citas Médicas**, desarrollada en React como parte del proceso de evolución del proyecto formativo. Su propósito es representar de manera visual y funcional los módulos principales del sistema, manteniendo coherencia con la lógica previamente desarrollada en Java y MySQL.

Actualmente, este front-end permite visualizar y simular procesos como el inicio de sesión, el registro de pacientes, la programación, cancelación y reagendamiento de citas, así como la consulta de historial, médicos y consultorios. En fases posteriores, este componente se conectará con el backend y la base de datos del proyecto principal.

## Funcionalidades principales

- Inicio de sesión prototipado
- Registro visual de pacientes
- Programación de citas
- Cancelación de citas
- Reagendamiento de citas
- Consulta de historial de citas
- Visualización de médicos en la red
- Visualización de consultorios principales
- Navegación modular mediante menú lateral

## Tecnologías utilizadas

- JavaScript
- CSS
- Visual Studio Code
- Git y GitHub
- Codex como apoyo para pulir y estructurar el código

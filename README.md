# Sistema de Gestión de Citas V.01

**Autor:** Julian David Lopez G.

En este documento podrá encontrar la información general sobre mi proyecto.

## Descripción
Mi sistema es un prototipo funcional diseñado para gestionar citas médicas. Permite registrar pacientes, agendar citas, cancelar citas, reprogramar citas, consultar horarios disponibles e iniciar sesión mediante un login prototipado.

## Funcionalidades principales
- Registro de pacientes
- Inicio de sesión
- Programación de citas
- Cancelación de citas
- Reagendamiento de citas
- Consulta de historial de citas
- Consulta de horarios disponibles

## Tecnologías utilizadas
- Java
- Visual Studio Code
- XAMPP
- phpMyAdmin
- JDBC
- Codex como apoyo para pulir y estructurar el código

## Extensiones utilizadas
- Debugger for Java
- Test Runner for Java
- Gradle for Java
- Codex
- Language Support for Java by Red Hat

## Ejecución del proyecto
Para ejecutar el proyecto, se debe ejecutar directamente el archivo `Main.java`.

## Nota
Este proyecto fue desarrollado como un prototipo funcional en consola, conectado a una base de datos MySQL para validar la lógica de negocio y la persistencia de la información.
